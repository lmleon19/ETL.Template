# Examples

Ejemplos mínimos de uso de utilidades de `ETL.Common`.

Estos archivos no son ETL completos. Solo muestran patrones pequeños y reutilizables.

## Ejemplos disponibles

- `DescargarArchivo.md`
- `DescomprimirZip.md`
- `DetectarEncoding.md`
- `LeerCsv.md`
- `SqlBulkCopy.md`
- `EjecutarStoredProcedure.md`

El proyecto `src/ETL.NombreProceso` es una base casi vacía para crear un ETL real.
El flujo completo debe definirse cuando se cree cada proceso específico.
