# Catálogo inicial ETL.Common

| Utilidad | Estado | Responsabilidad futura |
| --- | --- | --- |
| `ResultadoOperacion` | Base inicial | Representar resultados exitosos o fallidos. |
| `ServicioDescargaHttp` | Esqueleto | Descargar archivos HTTP/HTTPS. |
| `ServicioZip` | Esqueleto | Descomprimir archivos ZIP. |
| `DetectorEncoding` | Esqueleto | Detectar encoding de archivos texto. |
| `ServicioCsv` | Esqueleto | Leer archivos CSV. |
| `ValidadorCsv` | Esqueleto | Validar estructura de CSV. |
| `ServicioSqlBulkCopy` | Esqueleto | Cargar datos a tablas Stage. |
| `ServicioProcedimientoAlmacenado` | Esqueleto | Ejecutar procedimientos almacenados. |
| `ServicioSistemaArchivos` | Esqueleto | Centralizar operaciones de archivos y carpetas. |
