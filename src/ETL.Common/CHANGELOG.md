# Changelog ETL.Common

## 0.1.0

- Se crea estructura inicial de la librería común.
- Se agregan esqueletos de servicios y documentación base.

## 0.2.0

- Se implementan utilidades genéricas de `ETL.Common`.
- Se agrega detección de encoding por BOM con fallback configurable.
- Se implementan operaciones comunes de archivos, ZIP, CSV, validación, descarga HTTP y SQL.
- Se agregan pruebas unitarias para utilidades locales.
- Se agrega validación de estructura básica para archivos CSV.
- Se agrega `ValidadorArchivo` para validaciones genéricas de archivos.
- Se agrega soporte para campos CSV con saltos de línea dentro de calificadores de texto.
- Se agregan servicios comunes para hash de archivo, fecha de proceso, rutas de trabajo, normalización de texto y mensajes estándar.
